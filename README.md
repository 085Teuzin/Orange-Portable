# 🍊 Orange Portable

> **A sua biblioteca definitiva de jogos CSO para o emulador PPSSPP.**

O **Orange Portable** é uma experiência web otimizada, leve e modular, projetada para quem busca rapidez e organização ao baixar jogos comprimidos (CSO). Tudo o que você precisa para o seu PSP, acessível direto pelo navegador.

## 🌐 Acesse Agora
👉 **[https://085teuzin.github.io/Orange-Portable](https://085teuzin.github.io/Orange-Portable)**

---

## ✨ O que há de novo?
* 🎨 **Temas Dinâmicos:** O app muda de cor e visual automaticamente em datas comemorativas (Carnaval, Natal, Halloween e datas especiais!).
* ♓ **Efeitos Exclusivos:** Sistema de partículas temáticas e efeitos visuais imersivos que reagem ao calendário.
* 🚀 **Performance Web:** Otimizado para rodar perfeitamente em qualquer navegador mobile sem necessidade de instalação.
* 🗄️ **Estrutura Modular:** Código separado em módulos (Database, Infobase e Datas) para um carregamento ultra rápido.

## ✨ Funcionalidades Principais
* 🎮 **Biblioteca Curada:** Mais de 100 títulos selecionados e testados para garantir compatibilidade.
* ❤️ **Sistema de Favoritos:** Marque seus jogos preferidos e tenha acesso rápido a eles (salvo localmente).
* 🔍 **Busca Inteligente:** Filtro instantâneo por nome e categorias conforme você digita.
* 📂 **Categorização Completa:** Organização por gêneros (Ação, Corrida, RPG, Luta, Simulação e mais).
* 📦 **Foco em CSO:** Prioridade para arquivos comprimidos, economizando espaço no seu cartão de memória.

---

## 🛠️ Como Usar
1. Acesse o link oficial pelo seu navegador preferido (Recomendado: Chrome ou Safari).
2. Utilize a barra de busca ou as categorias para encontrar seu jogo.
3. Clique no ícone de download e pronto!
4. **Dica:** Adicione o site aos seus "Favoritos" no navegador para um acesso ainda mais rápido.
