# 🍊 Orange Portable

O **Orange Portable** é um Web App moderno e otimizado para entusiastas de jogos portáteis. Ele funciona como um catálogo inteligente para o emulador PPSSPP, permitindo que os utilizadores explorem, filtrem e acedam a uma vasta biblioteca de títulos em formato CSO de forma rápida e intuitiva.

## ✨ Funcionalidades Principais

* 🔍 **Pesquisa em Tempo Real:** Localize qualquer jogo instantaneamente através da barra de busca inteligente.
* 🗂️ **Filtros por Categoria:** Navegação simplificada por géneros (Ação, RPG, Desporto, Corrida, Luta, etc.).
* 📊 **Ficha Técnica Completa:** Veja detalhes como tamanho do ficheiro, data de lançamento, tempo estimado de jogo e nível de compatibilidade.
* 🎲 **Modo Aleatório:** O botão "Me Surpreenda" escolhe um título ao acaso para quando estiver indeciso.
* 🔃 **Ordenação Avançada:** Organize a sua lista por ordem alfabética, por tamanho ou foque apenas em jogos traduzidos (PT-BR).
* ❤️ **Sistema de Favoritos:** Guarde os seus jogos preferidos na sua lista pessoal (armazenamento local).
* 🆘 **Orange SOS:** Sistema integrado para reportar links quebrados ou solicitar novos títulos.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido com foco em performance e leveza, utilizando:

- **HTML5 & CSS3**: Estrutura robusta e estilização moderna.
- **Tailwind CSS**: Estilização via utilitários para uma interface responsiva e elegante.
- **JavaScript (ES6+)**: Lógica dinâmica para manipulação do DOM e gestão do estado dos filtros.
- **LocalStorage**: Para retenção dos dados de favoritos do utilizador sem necessidade de base de dados externa.
